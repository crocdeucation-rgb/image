2딱 하나만 기억하세요

담당자 빼고는 전부 "내용 칸에 글자만 쓰면" 시스템이 알아서 인식합니다.

담당자 지정만 직접 하고, 나머지는 내용 칸에 정해진 단어/형식으로 쓰면 끝.
예) 통화했는데 안 받으면 내용에 부재 → 상태가 자동으로 '부재/OFF'로 바뀜.
쓰고 나면 시스템이 처리 완료 표시로 내용 끝에 [✓]를 자동으로 붙입니다.

3내용 입력 규칙 (제일 중요)

왼쪽처럼 쓰면 → 오른쪽 상태로 자동 변환됩니다. 단어만 쓰면 되는 것과, 날짜·숫자 형식이 꼭 필요한 것을 구분하세요.

이렇게 입력결과주의

4방문 · 매출은 "형식"이 생명

이 둘은 단어만 쓰면 절대 안 잡힙니다. 날짜·시간·금액을 정확히 같이 써야 일정·매출 칸이 자동으로 채워집니다.

방문 6/29 5시반
방문 7/1 오전 11시
매출 6/29 3468
❌ 이러면 안 잡혀요

💡
매출 숫자는 노션 단위 그대로(예: 3468 = 346.8만원). 날짜는 6/29 형식, 시간은 5시반 · 오전 11시 · 오후 3시 형식이면 다 인식됩니다.

5수정할 땐 [✓] 뒤에 쓰세요

한 번 파싱되면 내용 끝에 [✓] 표시가 자동으로 붙습니다. 이게 '처리 완료' 도장이에요.

📌
같은 리드를 또 바꾸고 싶으면, 반드시 [✓] 뒤에 새 내용을 쓰세요.
예: 내용이 부재 [✓] 상태인데 방문이 잡혔다면 → 뒤에 이어서 방문 7/1 오후 3시 라고 쓰면 다시 인식됩니다.
[✓] 앞을 고치면 "이미 처리된 것"으로 보고 반영이 안 될 수 있어요.

6직접 해야 하는 것 (자동 안 됨)

대부분 자동이지만, 아래 두 가지는 손으로 처리합니다.

  1. 담당자 지정 — 신규 리드가 들어오면 담당자 칸에 본인 이름을 직접 선택하세요. (마케터끼리 정한 순서대로)
  2. 방문 취소 — 잡혔던 방문이 취소되면: ① 노션 방문 일정 칸의 날짜를 우클릭 → 삭제 → ② 상태를 '방문취소'로 직접 변경하세요. (이건 자동 안 됨)

7중복은 자동 처리됩니다

같은 전화번호로 또 신청이 들어오면 시스템이 알아서 '중복'으로 표시하고, 이전 기록을 내용 칸에 붙여줍니다.

전화번호가 같으면 자동으로 '중복' 상태가 되고, 내용 칸에 이전 신청 기록(날짜·담당자·상태)이 함께 표시됩니다. 따로 손댈 필요 없이, 이전에 어떤 상담이었는지 바로 확인만 하면 됩니다.

8우선연락 표시

꼭 빨리 연락해야 하는 리드는 우선 표시를 해두면 우선순위 탭에서 모아볼 수 있습니다.

  1. 내용에 우선연락 이라고 쓰면 → 📌 우선 체크박스가 자동으로 켜집니다.
  2. 또는 노션에서 📌 우선 칸의 체크박스를 직접 클릭해도 됩니다.
  3. 표시된 리드는 상단 '우선 순위' 탭에서 끼리끼리 모아볼 수 있어요.
‘우선 순위’ 탭에서 📌 체크된 리드만 모아본 화면입니다.

9필터로 골라보기

화면 상단에 두 종류의 필터가 있습니다. 고정 탭으로 빠르게 보거나, 직접 조건을 걸어 볼 수 있어요.

실제 노션 상단 화면 — 보라색·빨간색 영역을 참고하세요.
보라색 — 고정 필터 탭
빨간색 — 커스텀 필터
보라색 · 고정 탭
상단 탭(방문·오더·우선 순위·홀딩·일대일·부재)을 누르면 그 상태끼리만 모아서 봅니다. 미리 만들어둔 거라 클릭만 하면 돼요.
빨간색 · 커스텀 필터
우측 위 안테나(필터) 아이콘을 누르면 나옵니다. 상태별·유입경로별·담당자별·리드출처별 등 원하는 조건으로 직접 걸어 볼 수 있어요.
