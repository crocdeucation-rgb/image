02 — JUDGMENT LOG

판단의 기록

성공만 기록하지 않습니다. 위너처럼 보이는 것을 죽인 판단, 위너를 해부해 복제한 판단, 그리고 틀렸던 가설까지 — 세 개의 사건으로 일하는 방식을 보여드립니다.
CASE 1 · 2026.06
플랫폼 AI가 증액을 권한 캠페인을 폐기하다

MLST 무료검사 캠페인은 CPL ₩13,526 — 계정 내 상위권 효율이었습니다. 메타의 AI 어시스턴트는 구체적인 근거를 들어 예산 재배분을 권했습니다.

플랫폼 AI의 판단
  • "타 캠페인 대비 CPL 45% 저렴, 경쟁력 있는 수치"
  • "안정적인 리드 흐름 유지 중"
  • "어드밴티지+가 잠재고객을 유연하게 확보"
  • "성과가 저조한 캠페인의 예산을 이쪽으로 재배분할 것"
나의 판단
  • 상담팀 현장 보고 — "통화하면 학생만 온다"
  • CPM ₩5,000 미만 = 광고주 경쟁이 없는 값싼 오디언스에 대량 도달 중이라는 신호
  • 어드밴티지+가 35~57세 설정을 상한이 아닌 '제안'으로 취급 → 학생층으로 확장
  • 매출 데이터는 CRM(플랫폼 밖)에 있다 — AI는 ROAS를 볼 수 없다
CPL 위너였던 캠페인을 전량 폐기했고, 판단은 사중으로 검증됐습니다. ① 현장 — 상담팀의 "학생만 온다" ② 진단 — CPM 5천 미만 ③ 실측 — 메타 연령 리포트에서 리드 61%가 18–24세 ④ 확정 — 월 결산에서 DB 95건 중 오더 0건(전환율 0%). 직감에서 시작한 판단이 확정 데이터로 종결됐습니다. 이 사건으로 판정 기준을 CPL 단독에서 CPM(진단) + 리드품질 + ROAS(판정) 교차검증으로 바꿨습니다. 도구의 권고는 도구가 볼 수 있는 데이터의 한계 안에서만 유효합니다.
CASE 2 · 2026.05 → 07
위너를 감이 아니라 원리로 분해해 복제하다

노트릴스 영상은 CPL ₩6,990, 단일 소재로 4,107만원(11건)을 만든 압도적 위너였습니다. 정성껏 촬영·편집한 다른 소재들이 죽는 동안, 색감이 낮고 화질이 다듬어지지 않은 이 소재가 이겼습니다. 운이 아니라 원리를 찾기 위해 소재를 해부했습니다.

요소 ①
반박불가 물증
1년치 누적 노트 실물 — 카피가 아니라 증거. 학부모는 주장을 믿지 않고 물증을 믿는다.
요소 ②
Raw 질감
B급 색감·무보정이 광고 방어막을 해제. 잘 만든 광고는 광고처럼 보여서 스킵된다.
요소 ③
첫 3초 = 궁금증
메시지를 던지지 않고 질문을 남긴다. 완성된 주장보다 미완의 장면이 손을 멈춘다.

이 3요소를 설계 원칙으로 삼아 파생 소재를 제작했습니다 — 같은 물증(노트), 같은 질감, 같은 3초 구조. 결과는 재현됐습니다.

CASE 3 · 2026.05
틀렸던 가설 — 타겟을 걸면 타겟이 올 것이다

연령 타겟을 35~57세로 설정하면 학부모가 유입될 것이라 가정하고, 두 캠페인을 정성껏 만들었습니다. 둘 다 틀렸습니다.

비평준화 지역 저격
CPL ₩36,220 → ₩100,334 폭등 · CPM 5천 미만 · 오더 0 → 폐기
기초부족 학생 찾기
CPL ₩21,390 → ₩51,887 · 학생 DB만 유입 · 오더 0 → 폐기
타겟은 학부모로 걸었지만 소재가 학생을 불렀습니다. 메타 연령 리포트 실측 — 비평준화 캠페인 리드의 53%가 18–24세, 35세 이상은 23%에 불과했습니다. "학생을 찾습니다"라는 소재 언어에 학생이 반응했고, 알고리즘은 그 반응을 따라 값싼 학생층으로 확장했습니다. 결론 — 연령 설정이 아니라 소재가 오디언스를 결정한다. 이후 모든 소재를 학부모의 언어로 재설계했습니다.
세 사건이 수렴한 지점

CPM은 오디언스 품질의 진단 신호다

값싼 CPM은 광고주들이 경쟁하지 않는 값싼 층(학생)에 도달했다는 뜻이고, 비싼 CPM은 경쟁이 붙는 구매결정권자(학부모)에 도달했다는 신호입니다. 2개월치 집행 데이터 전수 백테스트에 이어, 메타 연령 리포트 실측으로 최종 확증했습니다.

CPM ₩20,000+ 소재
ROAS 4~15배
35세+ 리드 93~100%
CPM ₩10,000 미만 소재
전부 오더 0
리드 최대 61%가 24세 이하

단, CPM은 단독 판정 기준이 아니라 진단 신호로만 씁니다. 최종 생사는 리드 품질과 ROAS로 교차검증합니다 — 낮은 CPM이 미개척 오디언스일 가능성, 소재 초기효과일 가능성을 항상 반례로 열어둡니다.

04 — SYSTEM

판단을 시스템으로

좋은 판단도 수작업이면 나를 넘지 못합니다. 반복되는 판단은 전부 코드로 옮겼습니다 — 개발자 없이 직접.
전채널 리드 파이프라인
구글폼·메타·카카오를 API로 통합 — 신청 발생 즉시 노션 자동 업로드 + Slack 실시간 알림 + 신청자 웰컴 메시지 발송. 별도로 10분 주기 대사(reconcile)로 누락분까지 자동 복구해 유입 손실을 구조적으로 차단.
부재 알림톡 자동화
상담 부재 시 상태 변경만 하면 30분 내 카카오 알림톡 자동 발송. 상담팀의 후속 컨택 수작업 제거, 회수율 개선.
판정 가능한 데이터 구조
담당자별 DB→방문→세팅→방문취소→오더 풀퍼널과 캠페인·매체별 CPL·ROAS를 결제일 기준으로 집계하는 대시보드를 직접 구축. MLST 전환율 0% 같은 생사 판정이 가능한 건 이 구조 덕분입니다. 경영 보고는 링크 하나로.
AI 제작 파이프라인
영상 편집(무음 컷·자막·BGM)은 AI 에이전트로, 소재 제작은 HTML 에디터를 직접 코딩해 디자이너 없이 1080×1080 소재를 양산.
